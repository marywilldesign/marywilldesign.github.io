Well, Flip! - coptic bound book

print graphic design web dev

'Well, Flip!' harnesses p5.js to animate text falling and gathering itself up in a different form. Each frame was then extracted using Python and placed in InDesign for printing. The goal was to combine a traditional media format (a flipbook) with code as I wanted to explore creative coding.
