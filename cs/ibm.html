IBM - Internal Projects Site

UX Web Dev Graphic Design

This project aimed to restructure Modo's marketing site to simplify navigation, align branding with the mobile app, and reduce support inquiries from prospective members.

Roles: Content Strategy, Visual Identity, Web Design, Web Development, User Research, R&D

Timeline: Nov 2022 – Present

The Challenge

Modo’s marketing site had evolved into a ‘franken-site’ as many stakeholders updated content independently...

We needed to simplify the overall site experience, ensure branding was consistent with the mobile app, and streamline help content to reduce support inquiries.

Key Objectives

  • Refresh the website branding to match the Modo mobile app
  • Rethink page hierarchy, user flows, and site map
  • Evaluate and implement clear copy with consistent voice and tone
